# 🌈 Mood Focus Dashboard

> **Transform your mood into motivation with a personalized productivity dashboard.** ✨

## 📖 About the Project

The **Mood Focus Dashboard** is a creative frontend web application that adapts its interface based on the user's current mood. By selecting a mood, users receive personalized motivational quotes, productivity tips, and music suggestions, creating an engaging and positive experience.

This project is built using **HTML, CSS, and JavaScript** and demonstrates the fundamentals of responsive web development, DOM manipulation, and interactive UI design.

## 🎯 How It Works

1. Open the application in your browser.
2. Select your current mood.
3. The dashboard instantly updates with:

   * A personalized theme
   * A motivational quote
   * A music suggestion
   * A productivity tip
